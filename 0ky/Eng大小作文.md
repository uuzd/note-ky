# Eng大小作文

## 总结

1. 审题点：
 [ ] 通知还是邮件【格式】 → notice, email 
 [ ]  是不是熟人【P1】 → 陌生人要有自我介绍
 [ ]  有没有情感 【P1】
 [ ]  任务点涉及有几个要素【P2】→ 介绍（what），建议（how），邀请（why）
 [ ]  回信还是号召参与【P3】
 
### 第一段：
【寒暄】通知不用写 

```
I hope this email finds you well.
```
【自我介绍-陌生人】

```
I'm Li Ming, the organizer of the/this campus food festival.
```

【情感表达】,如果任务点里有一定要写

```
With my great/deep/high 
gratitude/frustration(upset/worry/concern)/apology/excitement(thrill),
```

【写信/通知目的】

```
the email/notice aims to do/is to do (转述任务点-把第三人称改为第二人称)
```

### 第二段：
**【总起句】**

```
When it comes to campus food festival, here are three/four points I would like to 
highlight.
```
三个要素：Firstly, Secondly, Thirdly
四个要素：Firstly, Secondly, Thirdly, Fourthly,

**【任务点关键词-invite, introduce】**

```
The event is scheduled for May 5 at Student Activity Center/via Tencent Meeting. Please 
ensure that you are available that day.
```

**【任务点关键词-arrangement, details】是什么活动**

真题: 2017 邮件 接受教授的邀请做一个演讲+介绍演讲的细节

```
(题干中的人物-you) 可以了解 (题干中的事件) 的起因和潜在影响。
You can learn about the causes and potential impact/influence/effect of ….
(题干中的人物-you) 还可以了解可以采取哪些行动来减轻此类事件的影响。
You also know which action should be taken to ease the impact of these events.
```

**【任务点关键词-invite, reason】活动的意义**

【官方认可】(活动) 已获得官方认可。

```
The event has gained official recognition from the authority.
```

【大众认可】它在多个社交平台上也享有广泛的知名度。

```
亮点表达：boast, be credited with
It also boasts/is credited with widespread popularity on multiple social platforms.
```
【对自己有帮助】您可以与那些有共同兴趣的人进行交流。

```
You can interact with those who have shared interests.
```

【任务点关键词-tip, plan, advice】怎么做

```
It is highly recommended that you could consult some experts in…/on….to gain access to 
key/core information.
```

【求助网友/找资料】如果我是你，我会在小红书/知网 cnki.net/ Wikipedia 上寻找(题干中
的事件)的相关经验/相关文献。

```
If I were you, I would refer to some relevant experience on Xiaohongshu.
If I were you, I would refer to some relevant literature on cnki.net/Wikipedia.
```

【要付诸于行动】一个切实可行的计划应该立即执行，并始终是您的首要任务。

```
A feasible/viable plan is expected to immediately carried out and it should be top of your 
agenda.
```
### 第三段：

**【个人类】**

如果您能考虑我的意见，我将不胜感激。有任何其他问题，请随时与我联系。

```
I would appreciate it if you could consider my opinion. Please call me at any time when 
you have any question.
```

【群体类-活动类】
如果您有兴趣参与，请扫描下方二维码获取更多信息并提交您的申请。

```
If you are interested in participating in it, please scan the QR code below to acquire 
more information and submit your application.
```
