### 概率

**古典概率**

```
1. [0-1]
2. 不可能事件概率 0
3. 必然事件概率 1
4. 事件A发生的概率 = 1-事件A不发生的概率
```

**独立性事件**

1. 一般独立性事件
 事件A 的发生对事件B的发生的概率没有影响

```
P(A*B) = P(A) * P(B)
```
独立性事件和非独立性事件的区别
> 摸球后,放不放回的问题

2. 伯努利模型

(1) A发生的概率为p, 在n次独立的实验中, A恰好发生 k的概率

```
Ck/n x (P) ^ k * (1-P)^(n-k)
```

(2) A 在第k次首次发生的概率

```
(1-P)^(k-1) * (P)
```

### 排 列 组 合 （五种思路与五种模型）


#### 五种思路

* 准确分类，合理分步（加法原理 or 乘法原理）
* 少量无思路，穷举归纳（枚举的从小到大定位法）
* 区分排列，还是组合（有顺序 or 无顺序）
* 特殊条件，优先处理
* 正难则反，除法消序（遇见“至少”、“至多”首先考虑反向做）
* 
```
特殊条件处理顺序：
1）优先处理确定性特殊条件
如：甲必须在C位；十位数必须为2
2）再处理非确定性条件
如：甲不可以在C位，十位数不可以为2
3） 若同时出现2个或2个以上非确定性特殊条件，
无所谓先后顺序，但要注意分类讨论
```
